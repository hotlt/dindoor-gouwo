狗窝

轻量级本地 SQLite 知识库 for OpenClaw AI 代理
干净简洁，快速高效，随用随存

🧹

自动数据清洗

存入时自动去除多余空行空格，压缩数据体积，保持内容整洁干净，减小数据库尺寸。

🔑

自动关键词提取

基于词频统计自动提取关键词，过滤中文停用词，不用手动一个个输入，省心省力。

🔍

FTS 全文检索

使用 SQLite 内置全文搜索引擎，搜索快速准确，数据多了也不卡。

📁

分类标签管理

支持给内容添加分类标签，可以按分类筛选查看，整理有条有理。

🚀 一键安装

在 OpenClaw 中直接安装使用：

claw skill install tianya-claw/dindoor-gouwo

或者通过 GitHub 安装：

skillhub install dindoor/dindoor-gouwo

💡 使用示例

# 添加内容，自动提取关键词
python3 skills/dindoor-gouwo/scripts/gouwo.py add "麟德智造3匹机组仅售3300元" "麟德智造,价格,机组" "产品"
# 搜索内容
python3 skills/dindoor-gouwo/scripts/gouwo.py search "价格"
# 查看完整内容
python3 skills/dindoor-gouwo/scripts/gouwo.py get 1
# 列出"产品"分类
python3 skills/dindoor-gouwo/scripts/gouwo.py list 产品
