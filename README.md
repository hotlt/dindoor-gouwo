# 🐕 dindoor-gouwo (狗窝知识库)

> **让你的 AI 助手拥有一个有序的"狗窝"，随存随取，永不丢失。**

---

## 🌟 为什么叫"狗窝"？

这个名字背后有一个真实的故事：

> 以前认真投入工作的时候，我喜欢在桌上堆放大量的资料、书籍、笔记。
> 同事们走进办公室，总会说："你这屋像个狗窝！"
> 
> 但正是这种方式，让我在需要什么资料时能马上找到，效率极高。
> 领导要什么，我随手就能拿出来。别人给的资料，我随手就放到桌上。
> 
> **"狗窝"不是脏乱差，而是有序的热闹 —— 是投入工作的痕迹，是效率的证明。**

**「狗窝」就是这样诞生的：让 AI 也能拥有一个属于自己的知识库，随手存放、有序管理、需要时马上找到。**

---

## ✨ 特性

- 🐕 **本地知识库** - SQLite 轻量存储，默认零依赖，隐私安全
- 🔍 **FTS 全文检索** - 毫秒级搜索，想找什么一呼即应
- 🤖 **智能分词** - 可选 jieba 分词，关键词提取更准确，支持专业术语
- 🏷️ **分类管理** - 给内容打标签井井有条
- 🔥 **热点优先** - 越常用的内容排序越靠前
- 🔄 **重复合并** - 相似内容自动检测，避免冗余
- 💾 **定时备份** - 数据安全，永不丢失

## 🚀 快速开始

```bash

"把瑞华制冷的电话存到狗窝"

"从狗窝查询一下瑞华的电话"

# 列出所有
"从狗窝列出所有内容"
```

## 📖 命令说明

| 命令 | 用法 | 说明 |
|------|------|------|
| `add` | `add "内容" [关键词] [分类]` | 添加内容，自动提取关键词 |
| `search` | `search "关键词"` | FTS 全文搜索，热点优先 |
| `get` | `get <id>` | 获取条目完整内容 |
| `update` | `update <id> "新内容"` | 更新已有内容 |
| `list` | `list [分类]` | 列出所有内容 |
| `delete` | `delete <id>` | 删除指定条目 |
| `stats` | `stats` | 查看统计信息 |
| `backup` | `backup` | 备份数据库 |

## 💡 适用场景

- 📦 **产品资料** - 产品参数、价格、卖点，随存随用
- 📞 **联系方式** - 客户电话、合作方信息，一个不落
- 📝 **工作笔记** - 会议记录、待办事项、灵感想法
- 📚 **知识积累** - 学习心得、经验总结、技巧沉淀

## 🏗️ 技术架构

- **数据库**: SQLite（零配置、零依赖）
- **全文检索**: FTS5（SQLite 内置，毫秒级响应）
- **语言**: Python 标准库，无需安装额外包
